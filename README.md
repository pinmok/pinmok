Django CMF is a lightweight, modular Content Management Framework for Django, designed to accelerate the development of web applications.  

- **Modular architecture**: Core modules are open-source, allowing easy customization and extension.  
- **Open source**: Core modules are released under [MIT License](), promoting community contributions.  
- **Commercial extensions**: Optional private modules can be developed separately for commercial purposes.  
- **Ready for production**: Includes content management, user management, and basic utilities out-of-the-box.  

This project is suitable for developers who want a flexible, maintainable foundation for building Django-based web applications.
